Section 1 — Problem Statement

Why MRI Edge Detection?

  • MRI images are 2D grayscale matrices where biological structures are encoded as varying proton densities
  • Tumors are hard to spot: Edges lack defined borders due to noise and low-contrast gradients
  • Mathematical model: image is a sampled function f(i,j) = I(i·h, j·h) + η(i,j) where η is additive noise
  • Goal: estimate gradient ∇I from discrete, noisy samples to isolate healthy vs. pathological tissue
Image as Sampled Function
f(i, j) = I(i·h, j·h) + η(i, j)
Gradient Magnitude (Edge Strength)
|∇I| = √( (∂I/∂x)² + (∂I/∂y)² )
Gradient Direction (Edge Orientation)
θ = atan2( ∂I/∂y, ∂I/∂x )

Section 3.3 — Mathematical Preliminaries

Finite Difference Approximations

Taylor expansion underpins every edge operator. The choice of scheme determines truncation error.

Forward Difference
f'(x) ≈ [f(x+h) − f(x)] / h
Error: O(h) — Roberts
Central Difference
f'(x) ≈ [f(x+h) − f(x−h)] / 2h
Error: O(h²) — Prewitt, Sobel
Second Derivative
f''(x) ≈ [f(x−h) − 2f(x) + f(x+h)] / h²
Error: O(h²) — Marr–Hildreth
Convolution as the Unifying Operation
(K ∗ f)(i, j) = Σ_p Σ_q K(p, q) · f(i − p, j − q)

Every finite difference stencil applied to an image is equivalent to a discrete convolution with kernel K

Section 5.2 — Ill-Conditioned Problems

Smoothing as Tikhonov Regularization

Numerical Differentiation Error Bound
E(h) ≤ h²/6 · |f'''(x)| + δ/h
  • As h → 0, the noise term δ/h diverges — raw differentiation is ill-conditioned
  • Gaussian smoothing (σ) acts as regularization, managing the bias–variance trade-off
  • Small σ: low bias, high noise sensitivity. Large σ: smooth edges but loss of localization
  • Optimal σ depends on the expected noise level of the MRI scanner
# Gaussian kernel: σ controls regularization strength def gaussian_kernel(size, sigma): k = np.zeros((size, size)) center = size // 2 for i in range(size): for j in range(size): d = (i-center)**2 + (j-center)**2 k[i,j] = np.exp(-d / (2*sigma**2)) return k / k.sum() # normalize
Key insight: Smoothing trades accuracy for stability. In medical imaging, moderate σ is preferred — false positive edges near tumor boundaries can be more harmful than slightly blurred true edges.
Section 5.3 — Variational Optimization

Canny Detector: Optimal by Design

Canny's Optimality Criteria (1986)
SNR = |∫h| / √∫h² Localization = |h'(0)| / √∫h'²
  • Calculus of variations: finds the optimal filter h(x) by simultaneously maximizing SNR and spatial localization
  • Optimal solution ≈ first derivative of a Gaussian
  • Non-maximum suppression: interpolation step to thin edges to 1-pixel width
  • Hysteresis thresholding: dual-threshold graph traversal to link edge fragments

Deriche · 1987 · Recursive IIR · O(1) per pixel
Rational Function Approximation
H(z) = N(z) / D(z) — Padé approximant
Implements Canny's optimal filter as an Infinite Impulse Response (IIR) filter. Treats the filter as a rational function (Padé) rather than polynomial (FIR). Achieves constant-time O(1) computation regardless of smoothing scale σ.
Sub-Pixel Refinement (Lindeberg 1998)
Quadratic fitting: x* = (g₋₁ − g₁) / 2(g₋₁ − 2g₀ + g₁) — localizes edge to sub-pixel precision by fitting parabola to gradient magnitudes. Scale-space zero-crossing: L²ₓLₓₓ + 2LₓLᵧLₓᵧ + L²ᵧLᵧᵧ = 0 requires 3rd-order finite difference stencils.
Section 5 — Extended Methods

Wavelet Edge Detection: Multiresolution Analysis

  • Haar DWT decomposes the image into approximation and detail sub-bands at multiple scales
  • Detail coefficients (horizontal, vertical, diagonal) directly correspond to local gradient structure
  • Adaptive threshold: edge pixels where |coefficient| > k · σ_noise (sensitivity coefficient)
  • Superior noise handling vs. fixed-kernel operators — noise separated from signal at multiple scales
  • Evaluated across k ∈ {0.5, 1.0, 1.5, 2.0, 2.5} to find optimal precision/recall balance
# Haar DWT adaptive edge detector class DWT: def get_adaptive_edges(self): coeffs = pywt.dwt2(self.image, 'haar') _, (cH, cV, cD) = coeffs magnitude = np.sqrt(cH**2 + cV**2 + cD**2) # Adaptive threshold: k * estimated noise sigma sigma = np.median(np.abs(cD)) / 0.6745 threshold = self.k * sigma edges = (magnitude > threshold).astype(np.uint8) return cv2.resize(edges, self.orig_size)
Insight: σ estimated via MAD/0.6745 — robust to outliers. Threshold.py sweeps k values and plots precision/recall curves.

Section 6 — Key Findings

What the Numbers Tell Us

Precision vs. Recall Trade-off
No Free Lunch
Higher k → more precise edges but lower recall. Lower k → more edges found but many are noise. k ≈ 1.5 maximizes F1 for brain MRI data.
Best Single-Pass Operator
Canny Wins on F1
Hysteresis thresholding and NMS give Canny the highest F1 among classical operators. Computationally simple but requires careful σ selection per image.
Multiresolution Advantage
DWT Best Noise Handling
Haar DWT separates noise from signal at multiple scales. Adaptive thresholding outperforms fixed-coefficient operators in low-contrast MRI regions.
Worst Performer
Roberts is Fragile
O(h) forward difference + no smoothing = noise amplification. Acceptable only on clean, high-contrast images. Unsuitable for clinical MRI.
Speed vs. Quality
Deriche is Fastest
O(1) per pixel regardless of σ. IIR recursion as Padé approximant achieves Canny-quality edges in real-time — ideal for live MRI guidance systems.
Error Metric Choice
Distance-Based TP
Radius r=2px matching avoids penalizing 1-pixel localization shifts. More appropriate than pixel-exact matching for noisy medical images.

Prof Requirement — Multi-Scale Edge Analysis

Why Multi-Scale Beats Single-Scale

  • Problem: a single σ or kernel size cannot simultaneously handle coarse tumor boundaries and fine tissue edges — they live at different scales
  • Single-scale operators (Roberts, Sobel, Prewitt) apply one fixed kernel → miss either fine detail or broad structure
  • Canny's σ partially addresses this but requires a manual per-image choice
  • Haar DWT decomposes image into approximation + horizontal / vertical / diagonal detail bands at each level — edge structure is visible at every scale at once
  • Adaptive threshold per scale: k · σ̂_noise where σ̂ = median(|cD|) / 0.6745 — automatically calibrated to each level's noise floor
Level 1 — Fine Scale
High-Frequency Details
Sharp tissue boundaries, thin vessel walls, fine tumor margins. Noisy but spatially precise.
Level 2 — Medium Scale
Structural Edges
Major tissue–tumor boundaries, ventricle outlines. Signal-to-noise ratio improves significantly.
Level 3 — Coarse Scale
Global Anatomy
Brain hemisphere boundaries, skull-tissue interface. Very stable, low noise, broad detection.
Result: DWT achieves the best noise handling of all tested methods because it does not commit to a single scale. The sensitivity sweep over k directly quantifies the precision–recall trade-off the instructor required — something impossible to show with a single fixed-threshold operator.
Section 7 — Conclusion

Numerical Analysis Unifies
Image Processing

The 60-year arc from Roberts (1963) to Haar DWT mirrors the intellectual history of numerical methods itself — from crude forward differences to variationally optimal, multiresolution designs.

Finite Differences
Forward → central → 2nd-order. Every kernel is a Taylor expansion truncated at a chosen order.
Regularization
Differentiation is ill-conditioned. Gaussian smoothing = Tikhonov regularization managing the δ/h blow-up.
Optimality
Canny used calculus of variations. Deriche used Padé approximants. Both find theoretically optimal solutions.
