Section 1 — Problem Statement

Why MRI Edge Detection?

  • MRI images are 2D grayscale matrices where biological structures are encoded as varying proton densities
  • Tumors are hard to spot: edges lack defined borders due to noise and low-contrast gradients
  • Mathematical model: f(i,j) = I(i·h, j·h) + η(i,j) where η is additive scanner noise
  • Goal: estimate ∇I from discrete noisy samples to isolate healthy vs. pathological tissue boundaries
Image as Sampled Function
f(i, j) = I(i·h, j·h) + η(i, j)
Gradient Magnitude — Edge Strength
|∇I| = √( (∂I/∂x)² + (∂I/∂y)² )
Gradient Direction — Edge Orientation
θ = atan2( ∂I/∂y, ∂I/∂x )

        fp = np.sum((self.detected == 1) & (dist_to_gt > self.radius))
        return tp / (tp + fp) if (tp + fp) > 0 else 0.0
Section 3.3 — Mathematical Preliminaries

Finite Difference Approximations

Taylor expansion underpins every edge operator. The choice of scheme determines truncation error.

Forward Difference
f′(x) ≈ [f(x+h) − f(x)] / h
Error: O(h) · Roberts
Central Difference
f′(x) ≈ [f(x+h) − f(x−h)] / 2h
Error: O(h²) · Prewitt, Sobel
Second Derivative
f″(x) ≈ [f(x−h) − 2f(x) + f(x+h)] / h²
Error: O(h²) · Marr–Hildreth
Convolution — The Unifying Operation
(K ∗ f)(i, j) = Σₚ Σ_q K(p, q) · f(i − p, j − q)

Every finite difference stencil applied to an image is a discrete convolution with kernel K

Section 5.2 — Discrete Wavelet Transform

Haar DWT: Multiresolution Analysis

  • The 1D DWT decomposes a signal into approximation coefficients c_j,k and detail coefficients d_j,k derived from a mother wavelet ψ(t)
  • In a 2D Haar transform the image splits into four subbands — LL (approximation), LH (horizontal edges), HL (vertical edges), HH (diagonal noise)
  • Adaptive threshold λ = kσ̂√(2 log N) derived from the HH subband via MAD: σ̂ = median(|HH|) / 0.6745
  • Sensitivity k swept over {0.5, 1.0, 1.5, 2.0, 2.5} — optimal k = 2.5 selected by F1 on training subset
  • Staircase artefact: diagonal subband separation reduces precision to 0.412 vs 0.458 for Sobel
# Haar DWT adaptive edge detector
class DWT:
    def get_adaptive_edges(self):
        coeffs = pywt.dwt2(self.image, 'haar')
        _, (cH, cV, cD) = coeffs
        magnitude = np.sqrt(cH**2 + cV**2 + cD**2)
        # Universal threshold: k × σ̂ × √(2 log N)
        N = self.image.size
        sigma = np.median(np.abs(cD)) / 0.6745
        threshold = self.k * sigma * np.sqrt(2 * np.log(N))
        edges = (magnitude > threshold).astype(np.uint8)
        return cv2.resize(edges, self.orig_size)
Subbands LL = blurred approximation · LH = horizontal edges · HL = vertical edges · HH = diagonal detail — used for noise estimation σ̂
Section 5.3 — Variational Optimization

Canny Detector: Optimal by Design

Canny's Optimality Criteria (1986)
SNR = |∫h| / √(∫h²)     Loc = |h′(0)| / √(∫h′²)
  • Calculus of variations: finds optimal h(x) by simultaneously maximising SNR and spatial localisation
  • Solution ≈ first derivative of a Gaussian
  • Non-maximum suppression: quadratic interpolation along gradient direction to thin edges to 1 px
  • Hysteresis thresholding: dual-threshold graph traversal links weak edge fragments to strong ones

Implements Canny's optimal filter as an IIR filter. Treats the kernel as a rational function rather than a polynomial (FIR). Achieves O(1) constant-time computation regardless of σ.

x* = (g₋₁ − g₁) / 2(g₋₁ − 2g₀ + g₁)
Localises edges to sub-pixel precision.

Section 5 — Extended Methods

Wavelet Edge Detection: Multiresolution Analysis

  • Haar DWT decomposes the image into approximation and detail sub-bands at multiple scales simultaneously
  • Detail coefficients (horizontal, vertical, diagonal) directly encode local gradient structure
  • Adaptive threshold: edge where |coefficient| > k · σ̂_noise
  • Noise estimate: σ̂ = median(|cD|) / 0.6745 — robust MAD estimator
  • Evaluated over k ∈ {0.5, 1.0, 1.5, 2.0, 2.5} to plot the precision–recall curve
# Haar DWT adaptive edge detector
class DWT:
    def get_adaptive_edges(self):
        coeffs = pywt.dwt2(self.image, 'haar')
        _, (cH, cV, cD) = coeffs
        magnitude = np.sqrt(cH**2 + cV**2 + cD**2)
        # Adaptive threshold: k × estimated noise σ
        sigma = np.median(np.abs(cD)) / 0.6745
        threshold = self.k * sigma
        edges = (magnitude > threshold).astype(np.uint8)
        return cv2.resize(edges, self.orig_size)
Insight σ̂ via MAD/0.6745 is robust to outlier edges. threshold.py sweeps k and plots full P/R/F1 curves.

Section 6 — Key Findings

What the Numbers Tell Us

Precision vs. Recall
No Free Lunch

Higher k → more precise but lower recall. k ≈ 1.5 maximises F1. The sweep makes this trade-off explicit and measurable.

Why Canny underperforms
NMS Hurts on MRI

Non-maximum suppression thins edges to 1 px. Diffuse MRI tumour boundaries get suppressed, dropping recall to 0.577 vs 0.763 for Sobel.

Unexpected result
Simple Operators Win

Roberts, Sobel, and Scharr tie at F1 = 0.556 — the simplest methods outperform every sophisticated algorithm on this dataset.

Extreme trade-off
LoG: Recall = 0.982

Zero-crossings detect nearly every true boundary pixel, but precision collapses to 0.183. F1 = 0.305 — last place.

Speed vs. Quality
Deriche is Fastest

O(1) per pixel regardless of σ. IIR Padé approximant achieves Canny-quality in constant time — ideal for live guidance.

Sub-pixel insight
Highest Precision: Sub-Pixel

Polynomial and Lindeberg methods achieve precision 0.502–0.504 — the best in the suite — but sacrifice recall. Best when boundary position accuracy is the priority.

Section 5.2 — Multi-Scale Analysis

Why Multi-Scale Beats Single-Scale

  • The problem: a single σ or kernel size cannot handle coarse tumour boundaries and fine tissue edges simultaneously — they live at different scales
  • Single-scale operators (Roberts, Sobel) use one fixed kernel and miss either fine detail or broad structure
  • Canny's σ partially addresses this but requires a manual per-image choice
  • Haar DWT decomposes the image into approximation + H/V/D detail bands at each level — all scales at once
  • Per-level adaptive threshold k · σ̂ is automatically calibrated to each scale's noise floor
Result DWT is the only tested method that does not commit to a single scale. The k sweep makes the precision–recall trade-off visible — optimal k = 2.5 confirmed on held-out data.
Level 1 — Fine Scale
High-Frequency Details

Sharp tissue boundaries, thin vessel walls, fine tumour margins. High spatial precision, higher noise.

Level 2 — Medium Scale
Structural Edges

Major tissue–tumour boundaries, ventricle outlines. SNR improves significantly at this level.

Level 3 — Coarse Scale
Global Anatomy

Brain hemisphere boundaries, skull-tissue interface. Very stable, low noise, broad detection.

Section 7 — Conclusion

Numerical Analysis Unifies
Image Processing

The 60-year arc from Roberts (1963) to Haar DWT mirrors the intellectual history of numerical methods — from crude forward differences to variationally optimal, multiresolution designs.

Finite Differences

Forward → central → 2nd-order. Every convolution kernel is a Taylor expansion truncated at a chosen order.

Regularization

Differentiation is ill-conditioned. Gaussian smoothing manages the δ/h noise blow-up as h→0, trading localisation for stability.

Optimality

Canny used calculus of variations. Deriche used Padé approximants. Both derive theoretically optimal solutions.
