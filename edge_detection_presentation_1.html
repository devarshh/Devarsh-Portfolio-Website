Section 1 — Problem Statement

Why MRI Edge Detection?

  • MRI images are 2D grayscale matrices where biological structures are encoded as varying proton densities
  • Tumors are hard to spot: edges lack defined borders due to noise and low-contrast gradients. Differentiation is highly sensitive to noise η
  • Goal: a balance must be found to maintain localisation accuracy of subtle edges without loss of precision
  • Clinical relevance: while a medical practitioner visually analyses MRI scans for tumour diagnosis, researchers also employ automated anomaly detection techniques based on classical edge detection methods
  • Mathematical model: f(i,j) = I(i·h, j·h) + η(i,j) — estimate ∇I from discrete noisy samples to isolate healthy vs. pathological tissue boundaries
Image as Sampled Function
f(i, j) = I(i·h, j·h) + η(i, j)
Gradient Magnitude — Edge Strength
|∇I| = √( (∂I/∂x)² + (∂I/∂y)² )
Evaluation Metrics
Precision = TP/(TP+FP) · Recall = TP/(TP+FN)
F1 = 2·P·R / (P+R)

Matching rule We computed the pixel-wise distance between detected edges and the ground truth mask using OpenCV's distance transform. A detected pixel is counted as a True Positive if it lies within 2 pixels of any ground truth edge — avoiding unfair penalties for sub-pixel localisation differences.

Section 5.2 — Discrete Wavelet Transform

Haar DWT: Multiresolution Analysis

  • 2D Haar transform splits the image into LL (approximation), LH (horizontal), HL (vertical), HH (diagonal noise)
  • Adaptive threshold λ = kσ̂√(2 log N), where σ̂ = median(|HH|) / 0.6745
  • Optimal k = 2.5 selected by F1 on training subset — F1 = 0.499, Recall = 0.761
  • Analyses edges simultaneously at fine, medium, and coarse resolution
Staircase artefact Diagonal frequency separation creates stepwise approximation of diagonal edges — precision drops to 0.412 vs 0.458 for Sobel
Haar DWT · Sensitivity k Sweep · k = 2.5 optimal
Pros
  • Multi-scale — analyses all spatial frequencies simultaneously
  • Adaptive threshold calibrated to per-image noise floor
  • High recall (0.761) — matches first-order operators
  • Robust MAD noise estimator — not thrown off by outlier edges
Cons
  • Staircase artefact on diagonal edges — precision penalty
  • Lower precision (0.412) than Sobel (0.458)
  • Haar basis cannot represent diagonals continuously
  • Overall F1 = 0.499, below first-order operators
Section 5.3 — Variational Optimization

Canny Detector: Optimal by Design

Canny's Optimality Criteria (1986)
SNR = |∫h| / √(∫h²)    Loc = |h′(0)| / √(∫h′²)
  • Calculus of variations proves the optimal filter = first derivative of a Gaussian
  • NMS thins edges to 1 px — ideal for sharp edges, suppresses diffuse MRI boundaries
  • σ = 1.0 optimal — hysteresis ratio T_high/T_low = 3 held fixed
  • Recall drops to 0.577 vs 0.763 for Sobel due to NMS on diffuse gradients
Canny · σ Sweep · σ = 1.0 optimal · F1 = 0.487
Pros
  • Mathematically proven optimal among linear filters
  • Single-pixel-wide edges — high spatial precision
  • Hysteresis links weak fragments, reduces false positives
  • Best precision among non-sub-pixel methods (0.458)
Cons
  • NMS suppresses diffuse MRI tumour boundaries — recall = 0.577
  • σ = 1.0 blurs fine anatomical detail before detection
  • Lower F1 (0.487) than simple operators — optimal ≠ best here
  • Requires manual σ selection per dataset
Section 5.4 — Second-Order Methods

Marr–Hildreth LoG: Zero-Crossing Detection

5-Point Laplacian Stencil
L = [0,1,0 / 1,−4,1 / 0,1,0]  ·  LoG = ∇²(Gσ ∗ I)
  • Edges at zero-crossings of ∇²(G∗f) — inflection points in intensity
  • DoG (σ₁/σ₂ ≈ 1.6) approximates LoG via function approximation
  • σ = 0.7 optimal — zero-crossings produce naturally closed contours
  • Most extreme result: Recall = 0.982, Precision = 0.183, F1 = 0.305
LoG (Second-Order) · σ Sweep · σ = 0.7 optimal
Pros
  • Highest recall of all methods — 0.982 (finds nearly every edge)
  • Produces closed contours — ideal for delineating tumour regions
  • DoG approximation is separable and computationally efficient
Cons
  • Lowest precision (0.183) — fires on every noise fluctuation
  • Lowest F1 score (0.305) — last place in the suite
  • Zero-crossing criterion cannot distinguish true edges from noise residuals
  • Large σ merges nearby boundaries — scale sensitivity
Section 5.5 — Recursive IIR

Deriche Detector: Rational Function Approximation

IIR Filter as Padé Approximant
H(z) = N(z) / D(z) — rational function, not polynomial
  • Implements Canny's optimal filter as a recursive IIR — O(1) per pixel regardless of σ
  • α = 0.8 optimal — thresholds swept (0.25, 0.5, 0.75, 1.0, 1.5)
  • F1 = 0.388, Precision = 0.306, Recall = 0.573
  • Padé approximation preserves filter shape but introduces edge-strength attenuation
Deriche · α Sweep · α = 0.8 optimal · F1 = 0.388
Pros
  • O(1) per pixel — only linear-time method in the suite
  • Constant computation regardless of smoothing scale σ
  • Canny-quality design — derived from the same optimality criteria
Cons
  • Edge-strength attenuation from Padé approximation reduces precision
  • F1 = 0.388 — 8th of 9 in the suite
  • Lower recall (0.573) than first-order operators (0.763)
  • IIR approximation introduces ringing artefacts near strong edges
Section 5.6 — Sub-Pixel Methods

Sub-Pixel Localisation: Interpolation & Scale-Space

Polynomial Sub-Pixel · T = 0.5 optimal · F1 = 0.476
Quadratic Interpolation
x* = (g₋₁ − g₁) / 2(g₋₁ − 2g₀ + g₁)

Fits parabola to three consecutive gradient magnitudes. Localisation error reduced from O(h) to O(h²).

Lindeberg Scale-Space · T = 0.5 optimal · F1 = 0.434
Scale-Space Edge Condition
Lx²Lxx + 2LxLyLxy + Ly²Lyy = 0

Third-order stencils, 5-pixel support, O(h²) error. Adapts scale automatically to local image structure.

Pros
  • Highest precision in the suite — Poly: 0.502, Lindeberg: 0.504
  • Sub-pixel accuracy — best for measuring boundary position
  • Lindeberg adapts scale automatically to local structure
Cons
  • Sacrifices recall — Poly: 0.528, Lindeberg: 0.441
  • Polynomial method assumes parabolic gradient profile — fails on noisy/blurred regions
  • Lindeberg requires higher-order stencils — computationally expensive
Section 4 — Mathematical Preliminaries

Finite Difference Approximations

Taylor expansion underpins every edge operator. The choice of scheme determines truncation error.

Forward Difference
f′(x) ≈ [f(x+h) − f(x)] / h
Error: O(h) · Roberts
Central Difference
f′(x) ≈ [f(x+h) − f(x−h)] / 2h
Error: O(h²) · Sobel, Scharr
Second Derivative
f″(x) ≈ [f(x−h) − 2f(x) + f(x+h)] / h²
Error: O(h²) · Marr–Hildreth
Convolution — The Unifying Operation
(K ∗ f)(i, j) = Σₚ Σ_q K(p, q) · f(i − p, j − q)

Every finite difference stencil applied to an image is a discrete convolution with kernel K. The optimal threshold T = 40 was confirmed experimentally across all first-order operators.
